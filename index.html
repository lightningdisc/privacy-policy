Lightning Privacy Policy

This is the Privacy Policy for Lightning. This document is for our Privacy Policy. This document can be changed at any time without any warning. By using Lightning, you agree to the document. Do not copy/playgarize/steal this document.



User Data

Your user data is not viewed by anyone. The only data that is stored is your warnings in a server, and your user ID for our economy system. You have the right to ask for your data to be deleted at any time. Upon request, it will be removed within 72 hours from request.



Guild Data

Guild data is stored. As in the guild ID, and some features with true and false like Flag Translations, Ticketing, auto moderation, etc. etc. You have the right to ask for your guilds data to be deleted at any time. Upon request, it will be removed within 72 hours from request.



Channel Data

We store only channel ID's. They are stored for ticket logging, automod logs, and the suggestion system. You have the right to ask for your channels data to be deleted at any time. Upon request, it will be removed within 72 hours from request.
